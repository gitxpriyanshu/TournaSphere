# 🎨 TournaSphere Frontend | Premium UI

The user interface layer for the TournaSphere tournament platform. Built with **React 19**, **Vite**, and **Tailwind CSS**, featuring a bespoke glassmorphic design system.

## 🚀 Development Mode
```bash
npm install
npm run dev
```

## 🏗️ Technical Stack
- **React 19**: Utilizing the latest concurrent rendering features.
- **Vite**: Ultra-fast HMR and build optimization.
- **Tailwind CSS**: Powering the glassmorphic and matrix design tokens.
- **Lucide React**: Premium vector iconography.

## 🌌 Key Modules
- **Bracket Matrix**: Dynamic grid-based tournament visualization.
- **Sporty Calendar**: Custom-engineered date/time picker (12h system).
- **Leaderboard Hub**: Real-time ranking visualizations.
